Presents a tivo-like GUI for browsing and monipulating an image-sequence archive.
The main class is org.nees.tivo.ImageRepositoryBrowser.
Supported functions operate on image sequences and are: view (scroll, play), clip, save (copy or clipped copy), delete, merge (two sequences), capture from an RBNB source.
currently unimplemented: push to RBNB (provide a source), convert to Mpeg3, (later) convert to Mpeg4

CVS
  • repository: cvs.nees.org:/disks/cvs/neesgrid
  • module name: tivo
ZIP file: To install, unzip into directory of your choosing

classpath (see bulid.xml and lib/) requires only: this source, commons-cli-1.0.jar and turbine.jar

ant:
  • ant (defaults to run)
  • ant run - compiles and runs
  • ant compile - just compile
  • ant jar - compile and build a jar (at build/lib/tivo.jar)
